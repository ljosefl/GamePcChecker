## [1.2.3] — 2026-05-03

### Добавлено

- Скрипт **`scripts/organize-publish-layout.ps1`**: в поставке каталоги **`data`**, **`temp`**, **`etc`**; **`CHANGELOG.md`** и **`github-update.example.json`** в **`etc\`**. **`publish-release.ps1`** по умолчанию собирает **single-file** self-contained, архив zip через **tar**; **`-MultiFile`** — папка с множеством dll. GitHub Actions вызывают тот же `publish-release.ps1`.

## [1.2.2] — 2026-05-01

### Исправлено

- Строка **«свободно на томе»** и проверка **«место под установку»** опираются на **том установки игры** (ручной путь или автоопределение Steam/GGG), а не на диск с максимальной свободной ёмкостью на ПК.
- Подпись в сводке: **«Свободно на томе установки игры»** при известном пути; если путь не найден — по-прежнему **«Макс. свободно на томе»**.

## [1.2.1] — 2026-05-01

### Исправлено

- **Тип диска (SSD/HDD):** ложное «игра на медленном HDD» для SSD. Win32 `MediaType` «Fixed hard disk media» общий для SSD и HDD — HDD больше не определяется только по нему; добавлены эвристики по модели (в т.ч. Kingston SHFS… и др.) и объединение данных **MSFT_PhysicalDisk** + **Win32_DiskDrive**.
- **Главное окно:** ширина по умолчанию и минимальная ширина увеличены; действия «Сканировать / Копировать / Сохранить» вынесены на отдельную строку под полем пути — без наложения на поле ввода и «Обзор».

### Изменено

- Кнопки: более плоский вид (`CornerRadius` 2, без границы у основного шаблона).

## [1.2.0] — 2026-05-01

### Добавлено

- Опциональное поле **«Папка с игрой»** и кнопка **«Обзор…»**: можно вручную указать каталог установки Path of Exile / Path of Exile 2, если автоопределение не сработало; блок «установка и диск» тогда ориентируется на **том этого пути** (классификация **SSD / NVMe / HDD** и свободное место).
- Файл **CHANGELOG.md** в составе публикации (рядом с `GamePcChecker.exe`) и регламент обновления версии в **RELEASING.md**.

### Изменено

- **GameInstallProbeService**: при непустом пользовательском пути поиск по Steam/стандартным папкам не используется для этой проверки; уточнены тексты при ошибке пути, отсутствии папки и при папке без типичных `PathOfExile*.exe` / `Client.exe`.

## [1.0.0] — начальная публикация

- Сканирование железа и софт-среды, отчёт и рекомендации под профили PoE / PoE 2.
- Стресс-тест GPU (Direct3D 11), учёт результатов в отчёте.
- Проверка обновлений с GitHub при наличии `github-update.json`.
- Экспорт отчёта в буфер и в файл.
