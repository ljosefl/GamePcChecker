# Game PC Checker

Настольное приложение для **Windows**, которое собирает сведения о железе и софт-среде и выдаёт **разбор пригодности ПК** под выбранный профиль игры. Из коробки поддерживаются **Path of Exile** и **Path of Exile 2**; оценки и рекомендации носят **эвристический** характер (не замена бенчмарку в самой игре).

**Репозиторий:** [github.com/ljosefl/GamePcChecker](https://github.com/ljosefl/GamePcChecker)

## Возможности

- **Сканирование ПК** — CPU (модель, ядра, частота), RAM, диски и свободное место, сетевые адаптеры, разрешение экрана, краткий «срез здоровья» системы (память, подкачка, схема питания и т.д. — без претензии на полный мониторинг).
- **Видеокарта(ы)** — вендор (NVIDIA / AMD / Intel), имя адаптера, **объём видеопамяти** дискретной GPU по **DXGI** (тот же путь, что и у драйвера; удобно для значений выше «старых» лимитов WMI).
- **Графические API** — проверка возможностей **Direct3D** и **Vulkan** (уровень функций / наличие поддержки для сводки в отчёте).
- **Рантаймы и зависимости** — эвристическая проверка: VC++ Redistributable, .NET Framework, WebView2, Desktop Runtime, заметка про Java; где возможно — ссылки на официальные установщики.
- **Отчёт и вердикт** — текстовая сводка: условные «оценки» CPU/GPU, вердикты по RAM/VRAM/диску/API, блок «что улучшить», советы по периферии и нагрузке на БП (без измерения розетки).
- **Экспорт** — копирование отчёта в буфер или сохранение в `.txt`.

## Стресс-тест GPU (синтетика)

- Отдельное окно с нагрузкой **Direct3D 11** и тяжёлым шейдером на активном GPU (**NVIDIA, AMD, Intel**).
- Регулировка интенсивности, опциональная нагрузка на все логические ядра CPU, опция **VSync**.
- **Графики** средней частоты кадров и температур (через [LibreHardwareMonitor](https://github.com/LibreHardwareMonitor/LibreHardwareMonitor)); мощность GPU — если драйвер её отдаёт; потребление блока питания целиком из ОС недоступно.
- После остановки результат может учитываться в основном отчёте (если уже был выполнен скан).

## Обновления приложения

При наличии файла `github-update.json` рядом с `GamePcChecker.exe` (см. `github-update.example.json`) приложение при запуске запрашивает **последний релиз** на GitHub и при более новой версии показывает ссылки на **страницу релиза** и на **скачивание** вложения (предпочтительно zip под Windows x64).

Альтернатива: переменная среды `GAMEPCCHECKER_GITHUB_UPDATE` в формате `владелец/репо`.

## Требования

- **ОС:** Windows 10/11 (x64).
- **Сборка из исходников:** [.NET SDK](https://dotnet.microsoft.com/download) с поддержкой целевого `TargetFramework` из `GamePcChecker.App.csproj` (сейчас указана строка Windows TFM для .NET 10).
- **Готовая сборка без установки .NET на целевой машине:** используйте **self-contained** публикацию (см. `RELEASING.md`) — в состав входят нативные библиотеки и рантайм.

## Сборка и запуск (разработка)

```powershell
git clone https://github.com/ljosefl/GamePcChecker.git
cd GamePcChecker
dotnet build .\GamePcChecker.sln -c Release
dotnet run --project .\src\GamePcChecker.App\GamePcChecker.App.csproj -c Release
```

## Публикация для пользователей

См. подробности в **[RELEASING.md](RELEASING.md)** — скрипт `scripts/publish-release.ps1`, артефакты в `artifacts/`, создание тега и релиза на GitHub.

## Ограничения (важно понимать)

- Это **не внутриигровой** FPS-тест и не замер реальной игры.
- Температуры и мощность зависят от драйвера/EC; на части конфигураций датчики могут отсутствовать.
- Оценки CPU/GPU — **условные шкалы** под каталог игр, не абсолютные баллы производительности.

## Зависимости (ключевые)

- UI: WPF + часть WinForms (host для DirectX-панели стресс-теста).
- Графика стресс-теста: Vortice (Direct3D 11 / DXGI / компилятор шейдеров).
- Датчики: LibreHardwareMonitorLib (MPL-2.0).
